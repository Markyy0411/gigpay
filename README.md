# GigPay: Decentralized Escrow Platform

![GitHub Repo Size](https://img.shields.io/github/repo-size/Markyy0411/gigpay)
![Languages](https://img.shields.io/github/languages/top/Markyy0411/gigpay)
![License](https://img.shields.io/github/license/Markyy0411/gigpay)

GigPay is a decentralized, Web3-native escrow platform designed to eliminate middlemen, reduce transaction fees, and provide instant settlement for the global freelance economy using the Stellar network and Soroban smart contracts.

## 🛠️ Core Competencies Demonstrated
*   **Web3 Development:** Integration with Stellar Network and Freighter wallet for secure, trustless transactions.
*   **Smart Contract Interaction:** Deployment and management of escrow contracts using Soroban.
*   **Full-Stack Development:** Frontend architecture with React/Vite, backend synchronization via Supabase.
*   **UI/UX Engineering:** Implemented secure, performant, and accessible interfaces (CSP, Clickjacking protection).
*   **Problem Solving:** Architected solutions for cross-border payment inefficiencies.

## 📂 Project Structure & Features
*   `contracts/`: Soroban smart contracts for trustless escrow management.
*   `src/`: Core frontend logic including components, context, and page management.
*   `supabase_schema.sql`: Database schema definition for real-time marketplace synchronization.
*   **Features:**
    *   Authentic Freighter wallet signatures for fund security.
    *   Dynamic cost-saving calculation widget.
    *   Custom Web3-native verification puzzles.
    *   Real-time marketplace updates via Supabase.

## 🚀 Getting Started
1.  **Clone:** `git clone https://github.com/Markyy0411/gigpay.git`
2.  **Dependencies:** `npm install`
3.  **Configure:** Create `.env` with Supabase credentials.
4.  **Database:** Execute `supabase_schema.sql` in your Supabase project.
5.  **Develop:** `npm run dev`

# GigPay: The Decentralized Escrow for Freelancers

**APAC Stellar Hackathon Submission**
**Track:** Payment & Consumer Applications

## Overview
GigPay solves the massive problem of cross-border freelance payments. Current platforms take 20% in fees and international wires take days to clear. GigPay allows clients to fund tasks using USDC in a Soroban Smart Contract escrow. Once the work is approved, the funds instantly settle in the freelancer's Stellar wallet. Zero middlemen, instant finality.

## Running Locally
1. `npm install`
2. `npm run dev`
3. Connect your Freighter wallet to interact with the Client and Freelancer portals.
